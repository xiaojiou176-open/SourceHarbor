# Security Policy

Please do not open public issues for suspected vulnerabilities.

Primary security intake:

- GitHub Security Advisories: <https://github.com/xiaojiou176-open/sourceharbor/security>

Include:

- a concise description
- impact assessment
- reproduction steps
- any affected commit, workflow, or deployment surface

This repository does not offer email-based private vulnerability intake.

## Supported Branch

- `main`

## Response Goal

Best-effort initial response target: within 5 business days.
