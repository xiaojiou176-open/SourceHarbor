# Sample Corpus

Everything in this folder is **sample data**.

It exists to help people understand what SourceHarbor looks like without pretending these files are current live production results.

Use these assets for:

- read-only playground surfaces
- docs examples
- landing-page illustrations
- internal demos

Do **not** use them as:

- remote proof for current `main`
- release proof
- live operator state
- evidence that a provider or secret-backed lane has already been validated
