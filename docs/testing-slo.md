# Testing SLOs

This page defines the default release-readiness thresholds for SourceHarbor.

## SLO Thresholds (Defaults)

API success rate

Green: `>= 100%`
Yellow: `>= 98%`
Red: `< 98%`

Worker completion rate

Green: `>= 80%`
Yellow: `>= 75%`
Red: `< 75%`

Digest delivery rate

Green: `>= 85%`
Yellow: `>= 60%`
Red: `< 60%`
