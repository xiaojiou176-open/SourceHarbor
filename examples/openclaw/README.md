# OpenClaw Example Surface

This directory holds the first public OpenClaw-shaped example assets for
SourceHarbor.

What is here:

- `skills/sourceharbor-watchlist-briefing/SKILL.md`

How to use it:

1. Read [docs/compat/openclaw.md](../../docs/compat/openclaw.md).
2. If you want the installable path first, use
   [starter-packs/openclaw/README.md](../../starter-packs/openclaw/README.md).
3. Start the SourceHarbor MCP surface or point OpenClaw at the HTTP API.
4. Copy or adapt the public skill example below instead of opening internal
   `.agents/skills`.

Honest boundary:

- this is a public compatibility example
- it is not a registry-published OpenClaw plugin package
- it is not a marketplace claim
